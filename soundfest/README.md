# Soundfest

![Soundfest Icon](icon.png)

Projet réalisé en fin d'année de première STI2D pour l'oral d'IT. Pour voir un aperçu du projet, ouvrez le diaporama `Presentation_Soundfest.pptx`.

Le programme est uniquement dans le fichier `micro.ino` réalisé sur une Arduino avec les composants listé dans le diaporama ci-dessus. Le programme peut sembler très mauvais, puisque c'est ma première fois avec du C++/Arduino.

Ce projet rempli ses exigences, dont une exigence écologique pour ralentir le dérèglement climatique.

## Fonctionnement

Le *Soundfest* est le nom du dispositif qui se tient en l'air, sur une corde entre la scène et la régie, qui permet d'afficher le niveau sonore à l'aide de 3 couleurs distinctives : Le vert, le orange et le rouge.

Lorsque le niveau sonore est trop élevé, il est recommandé de s'éloigner du *Soundfest* pour éviter potentiellement d'avoir des problèmes auditifs.

Par ailleurs, le *Soundfest* peut naviguer sur la corde pour afficher le niveau sonore d'une zone spécifique d'un festival ou d'un concert.

## Réalisation

Le projet *Soundfest* a été réalisé par l'équipe fictive *Teamfest* avec seulement 3 mois pour inventer des prototypes, développer des idées et réaliser un prototype "*final*", ce dossier n'inclus seulement la partie programmation que j'ai (Nathan) réalisé, puisque certaines autres parties n'ont pas été terminés à temps.

L'icône du *Soundfest* a été créé par moi et approuvé par l'équipe.
